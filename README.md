# Why is my PC doing it?

A small desktop app to see what your PC is doing.

It shows running programs, system usage and (later) network activity in a way that is easier to understand.

This is not meant to replace Task Manager or antivirus software.

## What it does

- shows running programs
- shows CPU and RAM usage
- shows storage usage
- gives simple explanations for common processes
- has a simple mode and an advanced mode
- shows basic hints if something looks off

## What it does not do

- not an antivirus
- does not remove malware
- does not upload anything

## Why

Sometimes your PC is slow or doing something in the background and it's not clear why.

This is just a tool to get a better idea of what's going on.
